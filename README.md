## Sobre mí

Vengo del mundo del diseño gráfico editorial, donde aprendí a cuidar la composición, la jerarquía visual, la tipografía y la comunicación clara. Actualmente estoy orientando mi carrera hacia el desarrollo web Full Stack, combinando mi base creativa con tecnologías modernas para crear productos digitales completos.

Me interesan especialmente los proyectos donde el diseño y el código trabajan juntos: interfaces web, experiencias interactivas, productos digitales, videojuegos, tecnologías inmersivas y soluciones que aporten valor real a las personas.

---

## Qué estoy haciendo ahora

- Aprendiendo y reforzando desarrollo web Full Stack, katas, aprendizaje y proyectos personales en mis repositorios.
- Creando proyectos personales para mejorar mi portfolio técnico.
- Explorando tecnologías frontend modernas como React, Next.js y Tailwind CSS, asi como la integración de flujos de trabajo con Inteligencia Artifical.
- Combinando diseño gráfico, UX/UI y programación para crear experiencias digitales más completas.
- Probando herramientas de inteligencia artificial aplicadas al diseño, la creatividad y el desarrollo web.
